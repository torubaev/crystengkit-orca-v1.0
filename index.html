CrystEngKit-ORCA
GitHub repository Documentation README

Crystal-engineering computation tools around ORCA and Multiwfn.

A practical GUI suite for preparing ORCA/Gaussian input files, running ORCA calculations, and generating ESP maps, NCI plots, QTAIM critical-point views, HOMO-LUMO diagrams, and intermolecular-interaction summaries.

View on GitHub Open documentation

Start here

Clone the repository, install the required Python packages, install ORCA and Multiwfn from their official sources, then start with the ORCA Input Builder.

ORCA input builder

Prepare calculation input files from CIF, XYZ, or existing ORCA input files, then run and monitor ORCA jobs.

ESP, NCI, and QTAIM

Generate and visualize electrostatic-potential maps, noncovalent-interaction surfaces, and QTAIM critical points.

Publication figures

Create consistent molecular graphics and supporting computational summaries for manuscripts and SI files.

Basic installation

Clone the repository git clone https://github.com/torubaev/crystengkit-orca-v1.0.git cd crystengkit-orca-v1.0
Install Python packages pip install numpy pyvista matplotlib periodictable gemmi Pillow
Run the installer/checker python install/install.py
Start the ORCA Input Builder python tools/Orca_input/orca_input.py
CrystEngKit-ORCA is distributed via GitHub under the repository license. ORCA and Multiwfn are external programs and must be obtained from their official sources.
